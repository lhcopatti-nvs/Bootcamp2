# Portfólio web

Este repositório contém uma página web estática construída com HTML5 e CSS3 puro. O layout adota uma estética moderna e minimalista em formato dark mode, focada em tipografia, legibilidade e organização espacial, dispensando o uso de fotos de perfil ou elementos visuais complexos.

## Estrutura do Projeto

O projeto é mantido em um formato de arquivo único para simplificar a visualização, hospedagem e edição.

* `index.html`: Contém toda a marcação estrutural HTML e as regras de estilo CSS embutidas na tag `<style>`.

## Funcionalidades e Características

* **Estética Moderna:** Paleta de cores baseada em tons escuros (slate/navy) com detalhes de destaque em gradiente (índigo e ciano).
* **Design Responsivo:** O layout e a tipografia se adaptam automaticamente a telas de dispositivos móveis e desktops através de media queries.
* **Zero Dependências:** Construído inteiramente do zero, sem a necessidade de bibliotecas ou frameworks externos (como Bootstrap ou Tailwind).
* **Variáveis CSS:** Uso de variáveis no escopo `:root` para facilitar a modificação global da paleta de cores.
* **Interatividade Sutil:** Efeitos de transição suave (hover) aplicados aos blocos de informação (como a seção de educação).

## Como Utilizar e Customizar

1. **Visualização:** Baixe o arquivo `index.html` e abra-o diretamente em qualquer navegador web.
2. **Edição:** Abra o arquivo em um editor de código-fonte de sua preferência (ex: VS Code).
3. **Customização de Cores:** Para alterar o esquema de cores, modifique os valores hexadecimais das variáveis declaradas em `:root` no início do bloco CSS.
4. **Atualização de Conteúdo:** Altere o texto contido nas tags estruturais (`<h1>`, `<h2>`, `<p>`, `<span>`) para inserir os seus dados profissionais, acadêmicos e resumo pessoal.

## Tecnologias Utilizadas

* HTML5
* CSS3
