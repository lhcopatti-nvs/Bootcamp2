# Single-Page Portfolio - LinkedIn Clone

Este repositório contém uma página web estática construída com HTML5 e CSS3. O objetivo do projeto é replicar a estrutura visual e de componentes de um perfil da plataforma LinkedIn em formato de portfólio de página única.

## Estrutura do Projeto

O projeto é mantido em um formato simples de arquivo único para facilitar a visualização e edição direta.

* `index.html`: Contém toda a marcação HTML estrutural e as regras de estilo CSS embutidas na tag `<style>`.

## Funcionalidades e Características

* **Design Responsivo:** O layout ajusta os elementos (como o tamanho da foto de perfil e margens) de acordo com a largura da tela do dispositivo (via `@media queries`).
* **Zero Dependências:** O código não utiliza frameworks externos (como Bootstrap ou Tailwind), operando exclusivamente com CSS puro.
* **Componentização Visual:** O layout é dividido em "cards" isolados, simulando o padrão de UI do LinkedIn:
  * Cabeçalho (Banner, Foto de Perfil, Informações de Contato e Status).
  * Seção "Sobre".
  * Seção "Educação".

## Como Utilizar e Customizar

1. **Execução:** Baixe ou clone o repositório e abra o arquivo `index.html` em qualquer navegador web.
2. **Edição:** Abra o arquivo em um editor de código (como VS Code). 
3. **Imagens:** Substitua a URL na tag `<img>` pela URL da sua imagem. O arquivo atual utiliza uma imagem hospedada no Imgur.
4. **Conteúdo:** Altere os textos dentro das tags `<h1>`, `<p>`, e `<div>` para refletir suas informações pessoais, resumo e histórico acadêmico/profissional.

## Tecnologias Utilizadas

* HTML5
* CSS3
